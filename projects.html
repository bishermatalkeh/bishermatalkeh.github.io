Class assignments, AI use cases, and projects from CE 895: AI in the AEC Industry. Each project includes a description of what it represents and why it is helpful, important, or relevant.

AI-Assisted Construction Specification Drafting

Assignment GenAI Completed

Description: Used ChatGPT and Claude to draft construction specifications for a hypothetical commercial building project. Compared AI-generated specifications against CSI MasterFormat standards to evaluate accuracy, completeness, and compliance.

Tools Used: ChatGPT (GPT-4), Claude, CSI MasterFormat reference

Why This Matters: Specification writing is one of the most time-intensive tasks in construction project delivery. This exercise demonstrated that generative AI can produce accurate first drafts in minutes, potentially saving engineers hours of work per project. However, the exercise also revealed that AI sometimes generates plausible but incorrect technical details — reinforcing the need for professional review.

AI for Construction Cost Estimation

Assignment ML Estimating Completed

Description: Explored how machine learning models can improve the accuracy of construction cost estimates by analyzing historical project data. Used a dataset of past construction projects to build a simple regression model that predicts project costs based on building type, square footage, location, and material specifications.

Tools Used: Python (scikit-learn), Excel, ChatGPT for code assistance

Why This Matters: Inaccurate cost estimates are one of the leading causes of construction project failures. Traditional estimating relies heavily on individual experience and manual quantity takeoffs. Machine learning can process thousands of historical data points to identify cost patterns that human estimators might miss, leading to more accurate bids and better project outcomes.

AI-Powered Drawing Review Checklist

Assignment GenAI Quality Completed

Description: Developed an AI-assisted approach to reviewing construction drawings by using generative AI to create comprehensive review checklists based on project type, building code requirements, and common design errors. Tested the approach on sample structural and architectural drawings.

Tools Used: Claude, Gemini, Standard building code references

Why This Matters: Drawing review is critical to catching design errors before construction begins. As a supervision engineer, I routinely reviewed drawings against specifications and codes. An AI-generated checklist ensures comprehensive coverage of review items and reduces the risk of overlooking critical details — especially for complex projects with multiple disciplines.
